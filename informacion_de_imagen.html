Código del Ejemplo

informacion_imagen.py 
import cv2

img = cv2.imread('gato.jpg')
print("Dimensiones:", img.shape) #imensiones de la imagen. El formato es: (alto, ancho, canales)
print("Alto:", img.shape[0])# representa el alto (número de filas) o la cantidad de píxeles verticales.
print("Ancho:", img.shape[1]) #indica el ancho (número de columnas) o la cantidad de píxeles horizontales.
print("Canales de color:", img.shape[2])

Glosario de Funciones

cv2.imread()
Función de OpenCV que lee una imagen desde un archivo. Soporta múltiples formatos como JPG, PNG, BMP, etc.
cv2.imshow()
Función de OpenCV que muestra una imagen en una ventana nueva. Requiere cv2.waitKey() para mantener la ventana abierta.
cv2.destroyAllWindows()
Cierra todas las ventanas de OpenCV que están abiertas. Útil para liberar recursos después de mostrar imágenes.
